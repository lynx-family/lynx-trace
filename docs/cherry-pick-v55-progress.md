# Cherry-pick to Perfetto v55.3 Progress

## Goal

Cherry-pick Lynx/trace related commits from `origin/develop/v50.x` onto Perfetto `v55.3`, one commit at a time, then push the resulting branch to `kanan`.

Current branch: `feat/trace_query_fuzzy_search_v55`

Current base: `v55.3`

Current in-progress upstream commit: none

## Required Per-Commit Checks

For every cherry-picked commit after conflict resolution:

1. `tools/run_presubmit --merge-base HEAD~1`
2. `./ui/build`

If either fails:

1. Investigate the concrete failure.
2. Fix only the current commit's issue.
3. `git commit --amend --no-edit`
4. Re-run both checks.

Notes:

- Earlier commits before the presubmit requirement was clarified only had `./ui/build`; they still need per-commit presubmit backfill before final push.
- `tools/run_presubmit --merge-base HEAD~1` is the repository-level per-commit check currently used for format, generated files, and other repo checks.
- `tools/format-sources` can fail locally if `.cargo/bin/rustfmt` is missing; use targeted formatters when the changed files are not Rust, then confirm with presubmit.
